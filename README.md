# 教师评分系统前台

> _一个基于 Vite + React 的展示教师评价的前端应用_

## ✨ 核心特性

- ⚡ **超快开发体验** - Vite 的即时服务器启动和热更新
- 🧩 **React 18 特性** - 并发渲染、Suspense 支持
- 📱 **完全响应式** - 适配移动端/平板/桌面设备
- 🌐 **国际化** - 支持多语言（i18n）
- 🧪 **功能** - 支持对教师进行打星评分以及评价

## 🚀 快速开始

npm run dev

### 前置要求

- Node.js `>=18.0.0`

### 安装依赖

```bash
npm install
# 或
pnpm install
```
